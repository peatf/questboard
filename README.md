# Questboard

Retro-styled weekly check-in app inspired by classic Mac OS 8/9 window chrome and JRPG command menus.

## Tech

- Vite
- React
- TypeScript
- GitHub Pages via GitHub Actions

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Lint

```bash
npm run lint
```
